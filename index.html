Bienvenida PWA

Esta aplicación utiliza Service Worker para demostrar dos estrategias de caché:

  • Cache Only: Para esta estructura base, navegación y estilos (App Shell).
  • Cache First, Network Fallback: Para las librerías de terceros (FullCalendar y Select2), así como para las CDNs de Tailwind y Google Fonts.

Navega a las otras secciones para ver las estrategias en acción.
